# ข้าวบ้านออนไลน์

เว็บ static สำหรับร้านขายข้าวออนไลน์ พร้อมหน้าเมนู โปรเปิดร้าน และฟอร์มสร้างข้อความออเดอร์

## เปิดใช้งาน

เปิด `index.html` ในเบราว์เซอร์ได้ทันที หรือใช้ GitHub Pages จาก branch `main`

## ไฟล์สำคัญ

- `index.html` โครงหน้าเว็บ
- `styles.css` ดีไซน์ responsive
- `script.js` รายการเมนูและระบบสรุปออเดอร์
- `assets/rice-shop-hero.png` ภาพ hero ที่สร้างสำหรับโปรเจ็กต์นี้
